Utility Module

This module contains utility functions used in the game to manage the display and handling of game-related data. It includes functions for clearing the screen, retrieving level files, and displaying the game grid with a delay for real-time updates.

Module Dependencies

Built-in Modules: os, subprocess, sys, time

Third-party Libraries: termcolor

Function(s)

utility.clear_screen() → None

Clears the terminal screen to give the effect of real-time updates, used for updating the game grid and other visual content that needs to clear the screen before displaying.

utility.degridder(grid: list[list[str]]) → None

Clears the terminal and displays the grid with a delay to simulate real-time movement.

utility.get_level_files() → list[str]

Returns a list of level files located in the ‘levels’ directory. The level files should be named starting with “level”, the level number, and ending with “.in”.

utility.print_game_title() → None

Prints the game’s title in the terminal with ASCII art.
