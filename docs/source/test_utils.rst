Utility Test
====================

This set of unit tests is focused on validating utility functions and methods
from different classes. Specifically testing grid manipulation, leaderboard 
functionality, and level progression.

Test Dependencies
-------------------

- `get_level_files()`, `clear_screen()`, and `degridder()` from `utility` module
- `Leaderboard` class from `leaderboard` module
- `LevelManager` class from `level_manager` module
