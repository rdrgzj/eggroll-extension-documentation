Scoring Test
====================

The tests are designed to verify that the game mechanics, 
such as egg counting, scoring, and score finalization, 
work as expected under different conditions. 

Test Dependencies
-------------------

- `clear_screen()` from `utility` module
- `MovementManager` class from `movement` module
- `Level` and `Player` classes from `game` module
