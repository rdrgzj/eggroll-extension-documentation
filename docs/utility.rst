Utility Module
==============

This module contains utility functions used in the game to manage the
display and handling of game-related data. It includes functions for
clearing the screen, retrieving level files, and displaying the game grid
with a delay for real-time updates.

Module Dependencies
-------------------

**Built-in Modules**: os, subprocess, sys, time

**Third-party Libraries**: termcolor
