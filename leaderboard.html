Leaderboard Module¶

Manages leaderboard functionality of the game

Module Dependencies¶

Built-in Modules: os, subprocess, sys, time

class leaderboard.Leaderboard(level: int)¶

Bases: object

Manages the leaderboard, storing and displaying high scores. This uses a .txt file per level where we can read and write to store high scores. The .txt files are in leaderboard directory.

display_scores()¶

Displays the leaderboard scores after each level.

Prints the leaderboard to the terminal, showing the rank, player name, and score for each player in the top 5.

load_scores() → list[tuple[str, int]]¶

Loads the leaderboard scores from the scores file.

Reads the leaderboard file, splits each line into name and score, and appends them to the scores list as tuples.

store_scores() → None¶

Stores the current leaderboard scores to the scores file.

Writes the player names and scores to the leaderboard file, overwriting the existing data.

update_leaderboard(name: str, score: int) → None¶

Updates the leaderboard with a new score.

Adds a new player name and score to the leaderboard, sorts the leaderboard by score in descending order, and keeps only the top 5 scores. Then stores the updated leaderboard (handled by another function).
