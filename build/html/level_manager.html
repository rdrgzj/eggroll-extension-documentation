Level Manager Module¶

This module defines the LevelManager class which handles the functionality related to managing game levels. It includes the ability to load level data from files, and determine the next level in a sequence.

This assumes level files are structured with the number of rows, total moves, and a grid representation of the level. The level files must also be in ‘.in’ format.

class level_manager.LevelManager¶

Bases: object

Manages level-related functionality including loading and transitioning levels.

get_next_level(current_level: str) → str | None¶

The method looks for the current level in the list of available levels and returns the path of the next level. If the current level is the last one in the sequence or not found, None is returned.

load_level(filename: str) → tuple[list[list[str]], int]¶

Reads a level file and initializes the grid and move count.

The level file is expected to be structured as follows: (1) The first line contains the number of rows in the grid. (2) The second line contains the total number of moves required. (3) Each subsequent line represents a row of the grid, where each character represents an element of the grid.
