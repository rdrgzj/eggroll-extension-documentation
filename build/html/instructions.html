Instructions Module¶

This module handles the functionality for displaying the game mechanics and instructions. It is called when the user selects option #3 from the main menu.

instructions.display_mechanics() → None¶

Clears the terminal, shows game title, and displays detailed game mechanics and instructions.

Once the instructions are displayed, the user is prompted to press ENTER to return to the main menu.
