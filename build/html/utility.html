Utility Module¶

This module contains utility functions used in the game to manage the display and handling of game-related data. It includes functions for clearing the screen, retrieving level files, and displaying the game grid with a delay for real-time updates.

utility.clear_screen() → None¶

Clears the terminal screen to simulate real-time grid updates.

utility.degridder(grid: list[list[str]]) → None¶

Clears the terminal and displays the grid with a delay to highlight changes.

The function prints each row of the grid with a slight delay of 0.2 seconds to make the update visually clear.

utility.get_level_files() → list[str]¶

Returns a list of level files located in the ‘levels’ directory.

The function looks for files that start with ‘level’ and end with ‘.in’ file extension.

utility.print_game_title() → None¶

Prints the game’s title in the terminal with ASCII art.
