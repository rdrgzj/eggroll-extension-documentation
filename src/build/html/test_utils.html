Utility Test

This set of unit tests is focused on validating utility functions and methods from different classes. Specifically testing grid manipulation, leaderboard functionality, and level progression.

Test Dependencies

  • get_level_files(), clear_screen(), and degridder() from utility module

  • Leaderboard class from leaderboard module

  • LevelManager class from level_manager module

Tests

test_movement.test_contd_moves() → None

Tests the move_until_blocked() method for the MovementManager class; to reproduce, either call an assertion using the method within this function or create a try-except block if expecting for exceptions.

test_movement.test_egg_coordses() → None

Tests the egg_coords(), egg_coords_b() and egg_coords_r() methods for the MovementManager class; to reproduce, call an assertion using the method/s within this function.

test_movement.test_single_moves() → None

Tests the singular move_down(), move_left(), move_up() and move_right() methods, as well as tests the unifying move_eggs() method for the MovementManager class; to reproduce, either call an assertion using the method within this function or create a try-except block if expecting for exceptions.
